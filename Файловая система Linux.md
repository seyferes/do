

| **Директория**   | **Предназначение**                                                                                                                                         |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `/etc`           | Все настройки системы и программ                                                                                                                           |
| `/var/`          | Переменные данные: **логи**, **базы данных**, **почта**                                                                                                    |
| `/home`          | Личные папки пользователей (документы, конфиги)                                                                                                            |
| `/bin` & `/sbin` | Основные команды (`ls`, `cd`, `cp`) и системные утилиты                                                                                                    |
| `/root`          | Домашняя папка суперпользователя                                                                                                                           |
| `/lib`           | Содержит **библиотеки** — файлы с кодом, который необходим приложениям во время выполнения                                                                 |
| `/media`         | Директория, куда автоматически **монтируются внешние накопители** (например, USB-устройства) при их подключении                                            |
| `/proc`          | Виртуальная директория, создаваемая при запуске системы. Она содержит информацию о **текущих процессах**, управлении памятью и конфигурации оборудования   |
| `/tmp`           | Хранение временных файлов                                                                                                                                  |
| `/dev`           | Содержит **файлы устройств**. Многие из них создаются во время загрузки системы; здесь находятся виртуальные диски и представления аппаратного обеспечения |
