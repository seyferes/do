
`grep` — это мощная и незаменимая утилита командной строки в Linux, предназначенная для поиска текстовых шаблонов в файлах. Она позволяет быстро находить нужную информацию, фильтровать вывод других команд и анализировать логи. Ниже представлены основные опции `grep` и полезные советы по его использованию.


| Команда                | Описание                                                         |
| ---------------------- | ---------------------------------------------------------------- |
| `grep "pattern" file`. | Базовый поиск строки в файле                                     |
| `grep -i`              | Игнорировать регистр (Case **I**nsensitive).                     |
| `grep -v`              | Инверсия: показать всё, кроме строк с шаблоном (**V**invert).    |
| `grep -r`              | Рекурсивный поиск во всех папках и подпапках (**R**ecursive)     |
| `grep -n`              | Показать номера строк (**N**umber)                               |
| `grep -c`              | Показать только количество найденных строк (**C**ount).          |
| `grep -A n / -B n`     | Показать `n` строк **A**fter (после) или **B**efore (до) находки |
| `grep -C n`            | Показать по `n` строк с обеих сторон.                            |

##### Pro-Tips

- **Пайплайны:** Чаще всего `grep` используется в связке с другими командами через "конвеер" (`|`). Например: `cat long_file.txt | grep "Critical"`.
- **Цветной вывод:** Если твой терминал не подсвечивает находки, добавь `--color=auto`. В современных дистрибутивах это обычно уже настроено через `alias`.
- **Поиск в папке:** Если нужно найти, в каком именно конфиге прописан старый IP, используй `grep -rn "192.168.1.1" /etc/`.

**Подводные камни:**

- **Кавычки:** Всегда бери поисковую фразу в кавычки, если в ней есть пробелы или спецсимволы (например, `grep "Connection refused"`).
- **Бинарные файлы:** Если `grep` случайно полезет в бинарный файл (картинку или архив), он может "сломать" вывод терминала странными символами. Используй флаг `-I` (заглавная i), чтобы игнорировать бинарники.
