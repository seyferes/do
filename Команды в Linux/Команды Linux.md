
## Базовые команды навигации

| **Команда** | **Расшифровка**         | **Предназначение**                                | **Флаги**                                                                                                                                                                                                                                  |
| ----------- | ----------------------- | ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `pwd`       | print working directory | Выводит полный путь к текущей директории/каталогу |                                                                                                                                                                                                                                            |
| `ls`        | list                    | Показывает содержимое текущей папки               | `-l` — выводит «длинный» список. размер файлов, дату создания и права доступа<br>`-a` — показывает **все** файлы (all), включая скрытые<br>`-h` — делает размер файлов «человекочитаемым» (human-readable), переводя байты в Кб, Мб или Гб |
| `cd`        | change directory        | Команды для смены директории/каталога             |                                                                                                                                                                                                                                            |
**Флаги можно комбинировать**, например `ls -lah`

## Просмотр содержимого файла

| ----------- | --------------- | ------------------------------------------------------- | ---------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

| `less`      |                 | Открывает файл в режиме чтения с возможностью прокрутки | _Навигация в less:_ стрелки для прокрутки, `q` для выхода. |                                                                                                                                                                                                                      |

## Поиск файлов

| **Команда** | **Расшифровка** | **Предназначение**                  | **Синтаксис**                             | **Примечания**                                                                                                                                                                                                                                                                                               |
| ----------- | --------------- | ----------------------------------- | ----------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `locate`    |                 | Это «быстрый поиск» по базе данных. |                                           | Если ты создал файл минуту назад, `locate` его не найдет, так как база еще не обновилась. Чтобы обновить её вручную, нужна команда `updatedb`<br>Ищет только по имени и полагается на базу данных, которая может быть неактуальной                                                                           |
| `find`      |                 |                                     | `find [где_искать] [критерии] [действие]` | `-name "pattern"` — поиск по имени (можно использовать маски, например `*.txt` для всех текстовых файлов).<br>`-type` — тип объекта: `f` для файлов, `d` для папок (directories).<br>`-mtime -7` — файлы, измененные (**modified**) за последние 7 дней<br>`-size +100M` — файлы размером более 100 Мегабайт |
**Флаги можно комбинировать**, например `ls -lah`

## Справочная информация

Команд в Linux тысячи, и помнить все флаги (`-r`, `-f`, `-name` и т.д.) невозможно. 
Поэтому в систему встроены «шпаргалки».

1. **`man`** (manual) — это полноценная книга по каждой команде.
    - _Как использовать:_ `man ls`. Откроется подробная справка с описанием всех флагов 
    - _Навигация:_ `j`/`k` или стрелки — вверх/вниз, `/слово` — поиск по тексту, `q` — выход.
2. **`--help`** — краткая справка, которая выводится прямо в терминал.
    - _Пример:_ `mkdir --help`. Это быстрее, чем читать весь `man`
